# 💫 About Me

**Junior Full Stack Developer** at **SmartB Solutions** (Malaysia), working on the Netforce ERP platform — a Tornado/XML-based stack with a React frontend. This is my longest and most hands-on role, where I build and maintain enterprise Accounts and Stock workflows, eCommerce integrations (Shopee, Lazada, TikTok Shop, Shopify, DHL, SPX Express, NinjaVan), and AI-powered features including OpenAI-driven sales order generation and a Bayesian decision-support system. I also handle server-side support and client configuration on custom domains.

Before SmartB, I delivered Django backends, REST APIs, automation pipelines, and web scraping solutions across multiple companies — including a Rasa chatbot backend with Celery task queues (Elite Chain), image processing and HRM systems (Nixaam), and data cleaning with machine learning at Karachi Urban Lab.

## 🎓 Academic & AI Research

MS Data Sciences graduate from IBA Karachi, with coursework in Machine Learning, Deep Learning, Probabilistic Reasoning, and Data Engineering. Academic projects include:

- **FastMig** — a data migration platform using evolutionary computation and LLMs
- **LLM-Based OCR Improvement** — semantic imputation on Azure Computer Vision output via OpenAI
- **ETL Pipeline** — Airflow-orchestrated star schema data warehousing
- **Trike AI** — reinforcement learning game agent
- **Persona-Based Decision Making** — multi-persona LLM feedback with probabilistic verification
